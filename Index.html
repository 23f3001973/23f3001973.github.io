Risk Calculation Engine


def calculate_var(portfolio_returns, confidence_level=0.95):
    """
    Calculate Value at Risk using historical simulation
    
    Args:
        portfolio_returns: numpy array of historical returns
        confidence_level: confidence level for VaR calculation
    
    Returns:
        float: Value at Risk as percentage
    """
    import numpy as np
    
    # Sort returns in ascending order
    sorted_returns = np.sort(portfolio_returns)
    
    # Calculate index for given confidence level
    index = int((1 - confidence_level) * len(sorted_returns))
    
    # Return VaR
    var = abs(sorted_returns[index])
    
    return var * 100  # Convert to percentage

# Example usage
portfolio_var = calculate_var(daily_returns, 0.99)
print(f"99% VaR: {portfolio_var:.2f}%")

Financial Mathematics

Black-Scholes Option Pricing Model

$$C = S_0 N(d_1) - K e^{-rT} N(d_2)$$

where:

$$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}$$

$$d_2 = d_1 - \sigma\sqrt{T}$$

Portfolio Return Calculation

$$R_p = \sum_{i=1}^{n} w_i R_i$$

Portfolio risk: $\sigma_p = \sqrt{\sum_{i=1}^{n} \sum_{j=1}^{n} w_i w_j \sigma_{ij}}$

These mathematical models form the foundation of our derivatives pricing and portfolio optimization algorithms. Our quantitative team has enhanced the traditional Black-Scholes model with machine learning corrections for improved accuracy.

-- High-frequency trading data pipeline
CREATE TABLE trading_positions (
    position_id BIGINT PRIMARY KEY,
    account_id VARCHAR(20) NOT NULL,
    symbol VARCHAR(10) NOT NULL,
    quantity DECIMAL(18,8) NOT NULL,
    entry_price DECIMAL(18,4) NOT NULL,
    current_price DECIMAL(18,4) NOT NULL,
    pnl AS (quantity * (current_price - entry_price)),
    timestamp_utc TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    
    INDEX idx_account_symbol (account_id, symbol),
    INDEX idx_timestamp (timestamp_utc)
);

-- Real-time P&L calculation
SELECT 
    account_id,
    SUM(pnl) as total_pnl,
    COUNT(*) as position_count
FROM trading_positions 
WHERE timestamp_utc >= DATE_SUB(NOW(), INTERVAL 1 DAY)
GROUP BY account_id
ORDER BY total_pnl DESC;
